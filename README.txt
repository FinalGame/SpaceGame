SpaceGame
=========
El proyecto esta dividido en diferentes paquetes. 
Los paquetes con clases principales son
	SERVER
	CLIENT
	
Para ejecutar el "Server" se tiene que ingresar al paquete
	/server/SpaceServer
	Ejecutar la clase SpaceServer

El server solo funciona en linea de comando y muestra el puerto
conectado va actualizando cada vez que se conecta un cliente.

Para ejecutar el "Client" se tiene que ingresar actualizando
	/client/SpaceClient
	Ejecutar la clase ClientSpace
	
El cliente preguntara en que IP desea conectarse (Si es la misma maquina
se puede poner localhost). Si encuentra la conexion se iniciara el juego.
